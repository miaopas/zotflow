# Release Notes 1.0.11

## New Features

### Auto-Disable Annotation Tools After Use

A new toggle in **Settings → General → Reader** — **Turn off note, text, and image annotation tools after each use** — makes the reader automatically revert to the pointer tool after each annotation is created. This avoids accidentally creating back-to-back annotations when you only meant to make one. The setting is **on by default**.

> Requires reopening the reader to take effect.

### Custom EPUB Font

A new **Ebook Font** field in **Settings → General → Reader** lets you set a custom font family for EPUB documents. Any valid CSS `font-family` value is accepted (e.g. `Georgia, serif` or `"Literata", serif`). Leave the field empty to use the book's own embedded font. The settings description text renders in the chosen font as a live preview.

> Applies to both Zotero library EPUBs and local vault EPUBs. Requires reopening the reader to take effect.
