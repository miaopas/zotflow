# Release Notes 1.0.16

## Features

- Added native link handling in the Zotero reader, so absolute links open externally while vault-style links resolve inside Obsidian.
- Added auto-purge for source notes of trashed Zotero items after sync.
- Source note templates now normalize template paths automatically, so missing `.md` extensions no longer break note generation.
- Templates now use a best-effort extracted year instead of relying on the first four characters of the stored date.

## Bug Fixes

- Fixed source note sanitizing so the key `path` is treated as a special case.
- Fixed annotation image cleanup so deleted annotations remove their image files correctly for both local and remote items.
- Fixed local PDF sidecar handling for hidden folders such as `.zotflow/sidecars`, including save, load, rename, and delete operations.
